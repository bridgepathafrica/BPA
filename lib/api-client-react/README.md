# @workspace/api-client-react — Generated React Query Hooks

Auto-generated TanStack Query hooks for the Bridgepath Africa API. Generated by [Orval](https://orval.dev) from `lib/api-spec/openapi.yaml`.

## Usage

```tsx
import { useListJobs, useGetDashboardStats } from "@workspace/api-client-react";

const { data, isLoading } = useListJobs({ limit: 10 });
```

## Regenerating

Run after any change to `lib/api-spec/openapi.yaml`:

```bash
pnpm --filter @workspace/api-client-react run generate
```

> **Do not edit files in `src/generated/` manually** — they are overwritten on every generate run.

## Auth token wiring

The generated hooks call the token getter function registered at app startup:

```ts
// artifacts/bridgepath/src/App.tsx
import { setAuthTokenGetter } from "@workspace/api-client-react";
import { getAuthToken } from "@/lib/auth";

setAuthTokenGetter(getAuthToken);
```

This keeps the generated layer decoupled from the frontend's auth implementation.
